Erstellen Sie ein Shell-Script mydelete, das als Parameter eine Liste von Dateien hat. 
Diese Dateien sollen nun gelöscht werden. Das Shell-Script fragt aber für jede Datei, ob sie wirklich gelöscht werden soll. 
Nur bei der Antwort Ja wird sie gelöscht.
Hinweis: Sie sollen hier das Shell-Script myquestion aufrufen.
