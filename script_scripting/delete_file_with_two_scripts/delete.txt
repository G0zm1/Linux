Schreiben Sie ein Shell-Script, mit dem Sie eine Datei löschen können. 
Die zu löschende Datei soll als Parameter beim Aufruf mitgegeben werden. Legen Sie noch ein zweites Script an, in dem Sie fragen, ob die Datei wirklich gelöscht werden soll.
Diese Übung soll Ihnen unter anderem zeigen, wie Scripte untereinander aufgerufen werden können.
